# Decisiones de producto

## 2026-07-16 - Fuente visual oficial AI Studio / Stitch

Google AI Studio / Stitch queda definido como fuente oficial de verdad visual para el SIG Izango 360.

El codigo exportado desde AI Studio es una especificacion de diseño implementada. No debe copiarse como base funcional dentro del SIG.

Reglas:

- El SIG real conserva Next.js, Supabase, rutas, permisos, reglas de negocio y datos reales.
- AI Studio aporta lenguaje visual, densidad, jerarquia, App Shell, composicion, tarjetas, tablas, toolbars, fichas 360 y estados visuales.
- No se migran Vite, Express, endpoints Gemini, mock data, estado local ni funciones demo.
- Los colores deben pasar por tokens semanticos del Design System.
- El verde principal oficial es `#00E676`.
