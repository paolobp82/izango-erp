# Design System Izango

El ERP utiliza una UX consistente con tarjetas blancas, bordes suaves, fondo claro, acentos verdes Izango y navegacion tipo operativa.

## Estado Actual

- Componentes UI existentes en `components/ui`.
- Componentes financieros en `components/finanzas`.
- Foundation y patrones reutilizables en `components/design-system`.

## Regla de Esta Fase

No se migran pantallas al design-system.
No se redisenan modulos.
No se modifican componentes existentes.

El Core Engine solo define contratos tecnicos y no contiene estilos ni logica visual.
